1.0 Introduction

1.1 Overview

Parking has become a big challenge in busy areas due to the limited parking spaces and the growing vehicle numbers. This will cause delays, frustrations, and inefficient use of time. To solve this problem, automated parking systems are being developed to provide a more efficient solution. However, many current systems lack features like seamless automation and easy customer access. The automated car parking system combines traditional and modern parking technologies to improve parking efficiency and user experience. In this project, there will be one floor with sensor lights to indicate available spaces and one floor with an automated parking system that uses lift and conveyor mechanism to handle vehicles. Users can book the parking slots and manage their parking time through a mobile app.

This system provides a modern solution to urban parking challenges, with features like automatic car plate scanning, cashless payment options, and optimized space usage. Therefore, this project aim to create a convenient and efficient parking system that saves time, reduces effort, and improves user satisfaction.
